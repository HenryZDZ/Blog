# 包豪斯风格博客重设计

## 概述

将现有 Flask 博客前端从「极简单栏」重设计为**包豪斯现代主义风格**，以几何构成、原色点缀、功能至上为设计语言。

## 设计方向

- **风格**：包豪斯现代主义（Bauhaus Modernism）
- **布局**：单栏阅读流，720px 最大宽度
- **配色**：亮色 + 暗色双模式
- **几何元素**：圆、方块、三角、叉号四种形状，仅用于 logo 和页脚签名，不作页面装饰
- **字体**：系统默认字体栈（`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto`），不加载外部字体
- **目标平台**：桌面端优先，移动端适配后续再做

## 站点名称

- **站名**：屏息之间
- **副标题**：Breathe, breathe in the air

## 配色系统

### 亮色模式

| Token | 色值 | 用途 |
|-------|------|------|
| paper | `#f8f6f0` | 页面底色 |
| ink | `#1a1a2e` | 正文、结构线 |
| red | `#e63946` | 强调色 / 装饰 |
| blue | `#2563eb` | 链接、引用线 |
| yellow | `#f4d03f` | 高亮 / 装饰 |
| green | `#10b981` | 标签补充色 |
| border | `#e5e5e5` | 细分割线 |
| muted | `#555` | 辅助文字 |

### 暗色模式

| Token | 色值 | 用途 |
|-------|------|------|
| bg | `#1a1a2e` | 页面底色 |
| text | `#f0ebe3` | 正文 |
| surface | `#2d2d44` | 卡片底色 |
| red | `#ff6b6b` | 强调色 |
| blue | `#60a5fa` | 链接 |
| yellow | `#facc15` | 高亮 |
| green | `#34d399` | 标签补充色 |

### 暗色模式切换

- 导航栏右侧放置切换按钮
- 使用 CSS 自定义属性 + `.dark` class 在 `<html>` 上切换
- 用户偏好存入 `localStorage`

## 几何元素规范

四种基础形状，固定尺寸容器（18×18px）对齐：

| 形状 | 颜色 | 使用位置 |
|------|------|----------|
| 圆 | red `#e63946` | logo、页脚 |
| 方块 | blue `#2563eb` | logo、页脚 |
| 三角 | yellow `#f4d03f` | logo、页脚 |
| 叉号 | ink `#1a1a2e` | logo、页脚 |

- 仅用于导航栏 logo 区和页脚签名
- 不在页面背景散布几何装饰
- 分页按钮间可用作小分隔符

## 页面设计

### 导航栏

- 左侧：四形 logo + 站点名（粗体 900），点击回首页
- 右侧：仅暗色模式切换按钮，无导航链接
- 底部 2px 黑色结构线
- 无搜索框

### 首页

- 站点描述「Breathe, breathe in the air」作为 intro
- 文章卡片：左侧 4px 原色竖线（红→蓝→黄→绿→黑 循环），白底，日期 + 标签 + 标题 + 摘要
- 标签：纯色填充，5 色循环（红/蓝/黄/黑/绿），视觉重心统一
- 分页：方块按钮 + 原色小几何分隔符
- 空状态：居中提示文字

### 文章详情页

- 标题区：大标题 + 日期 + 标签（红圆点分隔）
- 正文区：16px，行高 1.8
- 代码块：深色底 `#1a1a2e`，语法高亮使用原色系
- 行内代码：灰底 `#e8e8e8`
- 引用：左侧 4px 蓝色竖线，白底
- 链接：蓝色 + 下划线

### 评论区

- **匿名制**：无需输入昵称
- 昵称：从预设池随机分配（20 个，迷幻/太空主题）
- 头像：纯色圆形 + 昵称首字，背景色从原色系随机取
- 前端用 `localStorage` 保持同一会话内昵称和头像不变
- 输入框和按钮使用 2px 黑线风格
- 回复缩进显示，左侧蓝色竖线

### 标签页

- 标题：「标签: xxx」
- 文章列表复用首页卡片样式

### 搜索页

- 标题：「搜索: xxx」
- 结果列表复用首页卡片样式
- 空结果提示

### 关于页

- 单篇文章样式，显示 config 中的 ABOUT_TEXT

### 404 页

- 居中大标题 + 描述文字 + 最近文章列表

### 页脚

- 顶部 2px 黑色结构线
- 仅四形几何签名，无文字

## 标签配色方案

5 种纯色填充循环使用，不按标签语义固定颜色：

```
红 #e63946 → 蓝 #2563eb → 黄 #f4d03f → 黑 #1a1a2e → 绿 #10b981 → 循环
```

## 技术实现

- CSS 自定义属性（`:root` 和 `.dark` 双套变量）
- 纯 CSS 实现，不引入 CSS 框架
- 暗色模式通过 JS 切换 `document.documentElement` class
- 几何元素使用内联 SVG 或纯 CSS（border-radius / transform rotate）
- 语法高亮更新 pygments.css 与原色系协调

## 不在范围内

- 移动端响应式适配（后续再做）
- 后端改动（仅涉及模板和 CSS）
- 新功能新增
